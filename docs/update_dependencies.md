npm-check-updates (ncu)

Đây là tool mạnh nhất để update toàn bộ dependencies.

1. Cài đặt ncu
npm install -g npm-check-updates

2. Kiểm tra bản update
ncu

3. Update toàn bộ package.json lên phiên bản mới nhất
ncu -u


→ package.json sẽ được ghi đè toàn bộ version mới nhất.

4. Cài lại dependencies
npm install
